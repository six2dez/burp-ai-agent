    STEP A — Create DesignTokens.kt:

    Create directory src/main/kotlin/com/six2dez/burp/aiagent/ui/design/ and file DesignTokens.kt.

    Package declaration: package com.six2dez.burp.aiagent.ui.design

    Imports required: java.awt.Color, java.awt.Font, javax.swing.UIManager, java.awt.Insets.

    Define top-level object DesignTokens with three nested objects: Spacing, Typography, Colors.

    Spacing object — all Int constants (not computed):
      val xs = 4
      val sm = 8
      val md = 12
      val lg = 16
      val xl = 24
      val sectionPad = 8
      val formGridPad = 8

    Also expose four Insets constants (per UI-SPEC "Row inset constants"):
      val rowInsets get() = Insets(4, 8, 4, 8)
      val fieldInsets get() = Insets(4, 0, 4, 8)
      val fieldPairInsets get() = Insets(4, 0, 4, 12)
      val toggleRowInsets get() = Insets(8, 0, 4, 0)

    isDarkTheme at DesignTokens level (computed get):
      val isDarkTheme: Boolean get() { val bg = Colors.surface; val lum = (0.299*bg.red + 0.587*bg.green + 0.114*bg.blue)/255; return lum < 0.5 }

    Typography object — all computed get properties (baseFont and baseSize are private get).
    IMPORTANT: The headless fallback font size is 14 (matching UiTheme.kt line 55) to prevent
    font-size drift in headless tests:
      private val baseFont: Font get() = UIManager.getFont("Label.font") ?: Font("SansSerif", Font.PLAIN, 14)
      private val baseSize: Int get() = baseFont.size
      val sectionTitle: Font get() = baseFont.deriveFont(Font.BOLD, baseSize * 1.2f)
      val body: Font get() = baseFont.deriveFont(Font.PLAIN, baseSize.toFloat())
      val caption: Font get() = baseFont.deriveFont(Font.PLAIN, baseSize * 0.9f)
      val label: Font get() = baseFont.deriveFont(Font.BOLD, baseSize.toFloat())
      val mono: Font get() = UIManager.getFont("TextArea.font") ?: Font("Monospaced", Font.PLAIN, baseSize)

    Colors object — all computed get properties:
      val surface get() = UIManager.getColor("Panel.background") ?: Color(0xFFFBFF)
      val onSurface get() = UIManager.getColor("Label.foreground") ?: Color(0x1A1A1A)
      val onSurfaceVariant get() = UIManager.getColor("Label.disabledForeground") ?: Color(0x666666)
      val cardSurface: Color get() {
        val fromUi = UIManager.getColor("Table.background")
        if (fromUi != null) return fromUi
        val s = surface
        return Color((s.red - 10).coerceIn(0, 255), (s.green - 10).coerceIn(0, 255), (s.blue - 10).coerceIn(0, 255))
      }
      val border get() = UIManager.getColor("Component.borderColor") ?: Color(0xCCCCCC)
      val borderSubtle get() = UIManager.getColor("Separator.foreground") ?: Color(0xE0E0E0)
      val inputBackground get() = UIManager.getColor("TextField.background") ?: Color.WHITE
      val inputForeground get() = UIManager.getColor("TextField.foreground") ?: Color(0x1A1A1A)
      val primary get() = UIManager.getColor("Burp.primaryButtonBackground") ?: Color(0xD86633)
      val onPrimary get() = UIManager.getColor("Burp.primaryButtonForeground") ?: Color.WHITE
      val statusSuccess get() = UIManager.getColor("Burp.successColor") ?: Color(0x1B9E5A)
      val statusError get() = UIManager.getColor("Burp.errorColor") ?: Color(0xB3261E)
      val statusWarning get() = UIManager.getColor("Burp.warningColor") ?: Color(0xF57C00)
      val badgeNative: Color get() = if (isDarkTheme) Color(0x1E3A2C) else Color(0xE8F5EE)
      val badgeFull: Color get() = cardSurface

    CRITICAL: No `Color(0x......)` literal may appear in any color token property EXCEPT as a
    fallback after the `?:` operator on a UIManager.getColor() call, or in badgeNative/badgeFull
    which have no UIManager key (documented in UI-SPEC FLAG-01/02). The token layer must not
    hardcode primary color values as defaults in normal resolution paths. (badgeNative is a
    documented exception per UI-SPEC FLAG-01.)
