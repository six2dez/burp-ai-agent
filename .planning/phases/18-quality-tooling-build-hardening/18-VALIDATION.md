# Phase 18 — Validation Strategy

> Per-phase validation contract for feedback sampling during execution.

---

## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | JUnit Jupiter (JUnit Platform) via `useJUnitPlatform()` |
| **Config file** | `build.gradle.kts` `tasks.test` block (`-ea` assertions enabled) |
| **Quick run command** | `./gradlew test -PexcludeHeavyTests=true --no-daemon` |
| **Full suite command** | `./gradlew test --no-daemon` |
| **Estimated runtime** | ~60–120 seconds (full) |

---

## Sampling Rate

- **After every task commit:** Run `./gradlew test -PexcludeHeavyTests=true --no-daemon`
- **After every plan wave:** Run `./gradlew test --no-daemon`
- **Before `/gsd-verify-work`:** `./gradlew check --no-daemon` (includes detekt + ktlintCheck + tests) must be green
- **Max feedback latency:** ~120 seconds
