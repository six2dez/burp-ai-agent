## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Burp Scanner→doCheck | Burp's scanner calls doCheck() on the EDT or scanner thread; must not block |
| doCheck→executor | Async AI analysis runs on PassiveAiScanner.executor; result surfaced via siteMap().add() off-thread |
| siteMap().add()→Burp UI | Async findings posted to site map after AI completes; no synchronous risk |
