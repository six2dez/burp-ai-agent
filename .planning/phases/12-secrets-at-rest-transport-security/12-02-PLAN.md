<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Burp Preferences (disk) → AgentSettingsRepository.load() | Encrypted ciphertext crosses here; SecretCipher.decrypt() is the guard |
| AgentSettingsRepository.save() → Burp Preferences (disk) | Plaintext crosses here outbound; must always call cipher.encrypt() |
| migrateToSchemaV4() | One-shot read of plaintext, writes ciphertext; must not overwrite unless round-trip verifies |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-05 | Information Disclosure | migrateToSchemaV4 catch block | mitigate | Catch logs only the preference key constant (e.g. KEY_OLLAMA_API_KEY), never the raw value; enforced in action spec step 4 |
| T-12-06 | Tampering | Migration data loss (C2-2) | mitigate | Round-trip verify before plaintext overwrite; if verify fails, abort for that key and leave plaintext intact |
| T-12-07 | Denial of Service | decrypt failure corrupting all settings | mitigate | fail-soft per key — a corrupted ciphertext for one key returns "" for that key only; load() continues for remaining keys |
| T-12-08 | Elevation of Privilege | AgentSettings data class holding ciphertext | mitigate | Data class stays plaintext at runtime; encrypt/decrypt exclusively in repository I/O layer per architecture research anti-pattern 1 |
| T-12-SC | Tampering | npm/pip/cargo installs | accept | No package manager installs in this plan |
</threat_model>
