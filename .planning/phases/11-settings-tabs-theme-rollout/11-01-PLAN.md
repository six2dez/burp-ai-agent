    6. For "addVerticalFiller" at the end of each card panel: replace with addSpacerRow(panel, Spacing.sm).

    7. All ".background = UiTheme.Colors.surface" → ".background = DesignTokens.Colors.surface".
       All ".foreground = UiTheme.Colors.onSurface" → ".foreground = DesignTokens.Colors.onSurface".
       All ".foreground = UiTheme.Colors.onSurfaceVariant" → ".foreground = DesignTokens.Colors.onSurfaceVariant".
       All "UiTheme.Colors.primary" → "DesignTokens.Colors.primary".
       All "UiTheme.Colors.outline" → "DesignTokens.Colors.border" (UiTheme.outline is an alias for border).
       All "UiTheme.Typography.body" → "DesignTokens.Typography.body".
       All "UiTheme.Typography.label" → "DesignTokens.Typography.label".
       All "UiTheme.Typography.mono" → "DesignTokens.Typography.mono".

    8. Replace "LineBorder(UiTheme.Colors.outline, 1, true)" on JTextField/JPasswordField/JTextArea
       with calls to applyFieldStyle(field) or applyAreaStyle(area) from the design module.
