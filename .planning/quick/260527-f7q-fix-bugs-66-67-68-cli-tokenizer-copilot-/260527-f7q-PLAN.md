<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Burp extension -> external CLI process | Untrusted user-typed command string is tokenized into argv; quoting/escape errors can change which binary is launched. |
| Burp extension -> OpenAI-compatible HTTP endpoint | User-supplied API base URL + API key cross the boundary; response body crosses back into log/UI surfaces. |
| Extension -> debug log surface | Pre-flight payload preview must not leak prompt/message content into logs. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-quick-01 | Tampering | tokenizeCommand (AgentSupervisor) | mitigate | OS-aware tokenizer + unit tests cover Windows path + quoted-path cases so user input cannot be silently altered into a different argv. |
| T-quick-02 | Denial of Service | NonInteractiveCliConnection.send | mitigate | Explicit `redirectInput(/dev/null|NUL)` provides immediate EOF; combined with existing 60s wall-clock timeout this closes the Copilot interactive-menu hang vector. |
| T-quick-03 | Information Disclosure | OpenAiCompatibleBackend pre-flight debugLog | mitigate | `safeBodyPreview` deliberately excludes `json` and message bodies; only emits non-sensitive shape metadata (model name, counts, byte length). |
| T-quick-04 | Information Disclosure | OpenAiCompatibleBackend error message | accept | Up to 800 chars of provider response body are surfaced to logs/UI. Justification: required for diagnosability of #66; bodies from OpenAI-compatible 4xx responses are error metadata, not user payload. Truncation at 800 chars bounds exposure. |
| T-quick-05 | Spoofing | HeaderParser.withBearerToken | mitigate | Empty-token guard prevents sending malformed `Authorization: Bearer ` (which some servers interpret as unauthenticated and some as a spoof attempt); case-insensitive Authorization check prevents accidental overwrite. |
| T-quick-SC | Tampering | npm/pip/cargo installs | n/a | No new package installs in this plan — Kotlin/Gradle stack is locked and unchanged. |
</threat_model>
